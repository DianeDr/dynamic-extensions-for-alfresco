Data Dictionary Browser

The Data Dictionary Browser lets you view content models, types, aspects and associations.

View Data Dictionary

Note: currently you MUST install the json-rest-api for the Data Dictionary Browser to work. (The json-rest-api will be a standard component in the future.)

The Data Dictionary Browser is a proof-of-concept that explores rapid application development on top of Alfresco using modern JavaScript frameworks such as Ember.js.
